# PC5000A
PC5000A C++ CONTROL


W katalogu bin/Release/ jest gotowy program do obsługi przystawki IMUSB0403.

Program należy uruchomić w konsoli wiersza poleceń z parametrami

-p - numer portu COM
-n -
